# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute:** $0.10/user/month
- **Storage:** $0.05/user/month
- **Bandwidth:** $0.03/user/month
- **Total Cost per Active User:** 
  - **$0.10 + $0.05 + $0.03 = $0.18/user/month**

### Pricing Tiers
| Tier Name       | Price ($/mo) | Features                                                                 |
|------------------|--------------|--------------------------------------------------------------------------|
| **Basic**        | $19          | Intelligent retry timing, Basic dunning emails, Webhook error alerts    |
| **Pro**          | $49          | All Basic features + Advanced dunning strategies, Analytics dashboard    |
| **Enterprise**   | $99          | All Pro features + Custom integrations, Dedicated support, SLA guarantees |

### Customer Acquisition Cost (CAC) Range
- **Estimated CAC:** $50 - $100 per user

### Lifetime Value (LTV) Estimate
- **Average Monthly Revenue per User (ARPU):** 
  - Basic: $19
  - Pro: $49
  - Enterprise: $99
- **Average Retention Period:** 24 months
- **LTV Calculation:**
  - Basic: $19 * 24 = $456
  - Pro: $49 * 24 = $1,176
  - Enterprise: $99 * 24 = $2,376

### Break-even Users Count
- **Total Fixed Costs (e.g., marketing, development):** $10,000/month
- **Contribution Margin per User:**
  - Basic: $19 - $0.18 = $18.82
  - Pro: $49 - $0.18 = $48.82
  - Enterprise: $99 - $0.18 = $98.82
- **Break-even Users Count Calculation:**
  - Basic: $10,000 / $18.82 ≈ 532 users
  - Pro: $10,000 / $48.82 ≈ 205 users
  - Enterprise: $10,000 / $98.82 ≈ 101 users

### Path to $10K MRR
- **Target MRR:** $10,000
- **Pricing Tier Strategy:**
  - **Basic Tier:** 
    - Users Required: $10,000 / $19 ≈ 526 users
  - **Pro Tier:**
    - Users Required: $10,000 / $49 ≈ 204 users
  - **Enterprise Tier:**
    - Users Required: $10,000 / $99 ≈ 101 users

### Summary
To achieve $10K MRR, the most efficient path is through the **Enterprise Tier**, requiring **101 users**.
